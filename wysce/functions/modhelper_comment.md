# modhelper_comment  
&nbsp;  
# Description  
This function does nothing, it is generously used by the developers in a few spots in the game to explain how things work in some parts of decommpiled code. It's basically a replacement for a comment, since those don't save to decompilable code. 
&nbsp;  
# Arguments
### comment
The string that is to show up for modders in the decompiled code.
