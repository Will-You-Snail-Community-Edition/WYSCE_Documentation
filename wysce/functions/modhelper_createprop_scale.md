# modhelper_createprop_scale  
&nbsp;  
# Description  
Returns an OBJECT SCALE tool property to be added to an editor object in leveleditor_database_ini or modhelper_predatabase_override_me 
&nbsp;  
# Arguments
### f_min
The minimum scale of the object
&nbsp;    
### f_max
The maximum scale of the object
&nbsp;    
### f_scroll_steps
The amount that the slider in the editor's settings UI steps by
&nbsp;
