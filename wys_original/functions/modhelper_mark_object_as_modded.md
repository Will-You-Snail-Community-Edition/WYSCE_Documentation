# modhelper_mark_object_as_modded  
&nbsp;  
# Description  
Marks an object as modded, meaning any level using it will not be openable in the vanilla game. DO NOT USE THIS FUNCTION FOR WYS COMMUNITY EDITION! For vanilla mods, please call this function ALL custom object structs, or object structs you've modified to have extra properties that won't work on a vanilla game! 
&nbsp;  
# Arguments
### object_struct
The object struct you want to mark as modded.
